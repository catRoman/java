<h1 align="center">
WiZ 5.03
<img src="WiZ_files/zip-32-40t.png" width="40" align="absmiddle" height="40">
</h1>

<p>
WiZ is a graphical front end for <a href="http://infozip.sourceforge.net/Zip.html">Zip</a> and
<a href="http://infozip.sourceforge.net/UnZip.html">UnZip</a>; its primary author is Mike White.
WiZ currently runs only under 32-bit Windows (95/98/ME/NT/2000/XP/etc.),
although older releases also supported 16-bit Windows 3.x.  As with all
of Info-ZIP's software, WiZ is <a href="http://infozip.sourceforge.net/license.html">completely free</a>
and available as <a href="#Sources">source code</a>, too.  <b>Groovy</b>.

</p><p>
Note that a similar graphical port for the Macintosh, <a href="ftp://ftp.info-zip.org/pub/infozip/MACOS/"><b>MacZip</b></a>, is also
available.

</p><p>

<p>
</p><h2> Installation Instructions for Windows 3.x </h2>

<p>
(This section was provided by
<a href="http://www.users.bigpond.com/johncoom/">John Coombes</a>.)
<!-- also "mailto:johncoom@onaustralia.com.au" -->

</p><p>
</p><blockquote>
   <i>Important Note:</i>  <b>DO NOT double-click on <tt>wiz401x.exe</tt> while
   it is still in your download directory, as it is a self-extracting executable
   -- that is, it is NOT self-installing.</b>  If you do, it will extract itself
   into the download directory.
</blockquote>

<p>
Info-ZIP's 16-bit version of WiZ is for use with Windows 3.1 or 3.11.
It is compatible with PKZIP 2.04g, and it can zip, unzip and make
self-extracting EXE files.

</p><p>
All instructions assume that you are doing the installation from within
Windows 3.x and not directly from DOS.

</p><p>
</p><ol>
   <li>Open File Manager

      <p>
      </p><ol type="a">
         <li>Create a new directory for WiZ (say, <tt>C:\WIZ</tt>).
             To do this, click once on C:\ and then from the FILE menu
             choose CREATE DIRECTORY, type WIZ, and click OK

         <p>
         </p></li><li>Move (or copy) <tt>wiz401x.exe</tt> into the <tt>C:\WIZ</tt>
             directory.

         <p>
         </p></li><li>Double-click on <tt>wiz401x.exe</tt> and it will self-extract
             into the <tt>C:\WIZ</tt> directory.

         <p>
         </p></li><li>Now you should move <tt>wiz401x.exe</tt> onto a floppy or
             into a storage directory on your hard drive and keep
             it in case you need to re-install it later.
             If you copied (not moved) it into <tt>C:\WIZ</tt> you
             can delete the copy of <tt>wiz401x.exe</tt> -- it is not used
             when you run the WiZ program.
      </li></ol>

   <p>
   </p></li><li>Edit your AUTOEXEC.BAT and add the WiZ directory to the PATH statement.
      Typically it will look something like this:

      <p>
      </p><blockquote>
         <tt>PATH=C:\;C:\DOS;C:\WINDOWS;C:\WHATEVER</tt>
      </blockquote>

      <p>
      Add <tt>C:\WIZ</tt> to the end of it so it looks like this:

      </p><p>
      </p><blockquote>
         <tt>PATH=C:\;C:\DOS;C:\WINDOWS;C:\WHATEVER;C:\WIZ</tt>
      </blockquote>

      <p>
      You <i>must</i> put in a semicolon (;) just before you put in
      <tt>C:\WIZ</tt> .

      </p><p>
      </p><ol type="a">
         <li>You can edit AUTOEXEC.BAT by using NOTEPAD or any
